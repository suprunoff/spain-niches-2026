# Crypto Tax Spain — Quick Summary (1-page)

## Niche
**Crypto Tax + DAC8 + Modelo 721 Compliance for Spain**

Score: 8.56 | TAM: €30M | MVP: 6-8 weeks

## Problem
- DAC8 вступает в силу **1 янв 2026** (автоматический отчёт в Hacienda от бирж)
- Modelo 721 обязателен для €50K+ (дедлайн март, штраф €300-1500)
- **Конкуренты (Koinly, CoinTracking) не поддерживают полный DAC8 export + Modelo 721**
- Испанские инвесторы нужны локальные решения

## Market
- 25.14M крипто-пользователей в Испании (2026)
- 12% взрослого населения владеют крипто
- Объём рынка: €1.3B (2025) → €2.4B (2026), +85% CAGR
- Binance лидирует (60%+ рынка)

## Solution
MVP за 6-8 недель:
1. Синхронизация кошельков (Binance, Kraken, Coinbase API)
2. DAC8-compliant XML export
3. Modelo 721 auto-fill (PDF + Excel)
4. Tax obligation calculation (Spanish rules)
5. WhatsApp support (испанский)

## Unit Economics
- CAC: €18 (Twitter/X крипто-сообщество + Reddit)
- ARPU: €130/год (Koinly pricing: €49-179)
- LTV: €790 (при 10% churn) — хорошо
- **Проблема: реальный churn = 50% (сезонность)**

### Исправленная math:
- LTV при 50% churn = €260
- Ratio: CAC €18 → LTV €260 = 14.4x (маргинальное)
- **Решение: B2B focus** (accountants €500-2000/мес × lower churn)

## Risks (критичные)
1. **Сезонность (CRITICAL)** — спрос янв-март, потом -70% до янв следующего года
2. **Конкуренция от глобалов** — Koinly/CoinTracking добавят испанскую поддержку за 3-6 мес
3. **Регуляторный шок** — MiCA full enforcement с июля 2026 может изменить требования
4. **DAC8 misconception** — инвесторы могут подумать, что Hacienda уже знает всё (от DAC8)
5. **Нет AEAT API** — нужно генерировать PDF/XML для ручной подачи

## Recommendation

**✓ LAUNCH, but B2B-first strategy:**

1. **Position as B2B solution** (accountants, CPAs) — not B2C
   - B2C is seasonal, B2B is not
   - B2B ARPU is 5-10x higher
   - B2B churn is 10-15% (vs 50% for B2C)

2. **Timing: Dec 2025 / Jan 2026** (critical window before Koinly moves)

3. **MVP 6-8 weeks:**
   - Binance/Kraken/Coinbase sync
   - DAC8 XML export
   - Modelo 721 PDF/Excel generation
   - Email support (Spanish + English)

4. **Solve seasonality BEFORE launch:**
   - Add-on services for Apr-Dec (portfolio tracking, tax-loss harvesting)
   - Target accountants on LinkedIn + referral networks
   - Plan for annual plan conversion (customers pay Jan, use all year)

5. **Don't copy Koinly** — do Spanish-specific:
   - Full DAC8 support (theirs is partial)
   - Full Modelo 721 support (they use templates)
   - WhatsApp support
   - Spanish-first UI
   - Network with Spanish accountant associations
