## 1️⃣ Rental Insurance (8.5/10)

### Суть
**Comparador для страховки аренды** — как Rastreator, но только для renters + landlords.

### Почему это работает
- 14-16M людей арендуют
- Идеальная (новая) предложила June 2025 → доказано спрос есть
- Нет консолидированного сравнения
- 5.4% от ренты платят клиенты (= €50-100/мес на €1000)

### Конкуренты
Idealista (новый), Allianz, Generali, Rastreator (старый формат)

### MVP
1. Landing с калькулятором
2. Интеграция 5 страховщиков
3. Affiliate tracking
4. Запуск на 500 аккаунтов

### Метрики (Go/No-Go)
- CAC < €10 ✓
- LTV > €100 ✓
- 10% конверсия ✓

## 2️⃣ Non-Resident Mortgage (8.2/10)

### Суть
**Online mortgage broker для иностранцев**, покупающих недвижимость в Испании.

### Почему это работает
- 100K+ иностранцев ищут ипотеку в год
- Процесс долгий (6-8 нед) → онлайн = преимущество
- ЛУ вышед: US +20%, LatAm +40%, Poland top-5
- Нерезиденты платят 40% депозит, нужна помощь

### Конкуренты
Fluent Finance, Mortgage Direct, Barnes Madrid (все slow, offline)

### MVP
1. Pre-qualification (10 мин)
2. Интеграция 2-3 банков
3. Document management
4. Запуск на 50 аккаунтов

### Метрики (Go/No-Go)
- CAC < €100 ✓
- ACV = €1500-2500 ✓
- Payback < 3 мес ✓

## 3️⃣ VeriFactu SaaS (8.0/10)

### Суть
**Easy SaaS для VeriFactu компланка** — invoice tool + tax filing автоматизация.

### Почему это работает
- **100% адоптация** по закону к июлю 2027
- 5.7M компаний (autónomos + SME) в Испании
- **80-90% не готовы** к дедлайну (только 8-15% готовят решения)
- Штраф за несоблюдение €3000+
- Готовы платить €50-200/мес за спокойствие

### Конкуренты
EDICOM (дорого, enterprise), Holded (beta), Debitoor (слабо)

### MVP
1. Invoice creation + storage
2. Open Banking sync (95% банков готовы)
3. VeriFactu transmisión API
4. Onboarding 50 компаний

### Метрики (Go/No-Go)
- CAC < €30 (organic) ✓
- MRR 50 customers = €2-4K ✓
- Payback < 2 мес ✓

## 4️⃣ SME Open Banking API (7.8/10)

### Суть
**Accounting + cash flow forecast через Open Banking** для 2.5M SME.

### Почему это работает
- 95% банков в Испании готовы (через Redsys)
- 80% SME делают accounting вручную
- Open Banking adoption +20% в год
- Use cases: invoice financing, VAT reports, cash flow

### Конкуренты
BBVA Open Banking (big), Afterbanks (small), Yapily (European)

### Gap
Нет Spain-first, accounting-focused решения

### MVP
1. Bank sync (Open Banking API)
2. Auto-match с счётами
3. VAT reports
4. Cashflow forecast

## 5️⃣ Embedded Insurance (7.5/10)

### Суть
**Insurance-as-a-feature** — страховка при покупке товара, бронировании отеля, аренде.

### Почему это работает
- EU рынок = €40B потенциально
- Spain доля = €2-3B потенциально
- 70%+ е-комерс, но страховка не видна
- +2-5% на цену = люди платят

### Конкуренты
Klarna (слаба в Испании), нет специалиста

### MVP
1. Embedded widget в checkout
2. White-label с партнёром-страховщиком
3. Pre-underwritten policies
4. Запуск на 10 e-commerce сайтов

## Как выбрать первую нишу

1. **Контакты:** где у тебя есть теплые интро?
   - Если в недвижимости → Rental Insurance или Non-Res Mortgage
   - Если в бизнесе/accounting → VeriFactu или Open Banking
   - Если в e-commerce → Embedded Insurance

2. **Скорость:** готов ли ждать?
   - VeriFactu = 100% адоптация, но нужна лицензия (6-9 мес)
   - Rental Insurance = можно стартовать без лицензии (только comparador)
   - Mortgage = нужны банк-партнёры (3-6 мес)

3. **Capital requirements:**
   - Rental Insurance: €5-10K (landing + ads)
   - Non-Res Mortgage: €10-20K (ads + ops)
   - VeriFactu: €20-50K (development + tax agency relations)
   - Open Banking: €50-100K (multi-integration)
   - Embedded: €50-100K (compliance + insurance partner)
