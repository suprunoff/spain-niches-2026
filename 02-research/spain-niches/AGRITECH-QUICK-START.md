## ДВА ВЫХОДА НА NEXT 6 WEEKS

### 🥇 ПРИОРИТЕТ #1: PAC Subsidy Automation (Score 8.5/10)

**Что это:** SaaS для испанских фермеров, который автоматизирует подачу EU CAP (PAC) субсидий

**Почему быстро деньги:**
- 622 400 фермеров подают заявки каждый год
- Платят gestoras (агентства) 2-5% комиссии
- Нет конкурентов SaaS (только ручной труд)
- €4.875B бюджета ежегодно (государственный источник, не зависит от цикла)

**MVP (4-6 недель):**
```
Неделя 1-2: Interview 15 фермеров
            → Поймёшь процесс, боль, готовность платить

Неделя 3-4: Wizard web-form (20 вопросов)
            → Auto-validation против CAP requirements
            → PDF-экспорт + email
            → WhatsApp notifications

Неделя 5-6: 10 бета-юзеров
            → Feedback, iterate
            → Landing page + Google Ads (€500)
```

**Цена:**
- €250-400/год (малые фермы < 10 га)
- €400-600/год (средние 10-50 га)
- €600-1000/год (крупные > 50 га)

**LTV (5 лет):** €1250-5000 за фермера

**Риск:** 🟢 LOW (государственная программа, нет конкурентов)

**Выход:** 🚀 **GO** (Начни на неделе 1)

### 🥈 ПРИОРИТЕТ #2: Wine Label Compliance SaaS (Score 8.1/10)

**Что это:** Tool для генерации QR-кодов на вино (EU требует это)

**Почему быстро деньги:**
- EU Regulation 2021/2117: QR-коды обязательны с дец 2023
- 3000+ бодег в Испании не знают как реализовать
- Штрафы до €50K (мощный мотиватор)
- Нет локального конкурента (только иностранные: Scantrust, QRFox)

**MVP (3-4 недели):**
```
Неделя 1: Interview 15 бодег
          → Как реализовали QR? Сколько потратили? Готовы платить?

Неделя 2: Figma макет
          → Upload форма → Design preview → QR generator → PDF для типографии

Неделя 3: Code
          → React frontend + Node.js backend + QR library
          → PDF-export для типографии

Неделя 4: Beta на 10 бодег
          → Feedback, iterate
```

**Цена:**
- Freemium: 1 label/месяц (бесплатно)
- Starter: €200/год (до 50 SKU)
- Pro: €500/год (200+ SKU)

**Риск:** 🟡 MEDIUM (регулирование обязательно, но TAM меньше чем PAC)

**Выход:** 🚀 **GO** (Начни на неделе 1)

## ВАЛИДАЦИЯ: ЧТО ДЕЛАТЬ НА НЕДЕЛЕ 1

### Для PAC:

**Кого найти:**
```
LinkedIn groups:
  - "Agricultores España"
  - "Jóvenes Agricultores"
  - COAG / ASAJA / UPA страницы

WhatsApp:
  - "cooperativas agricolas españa"
  - Позвони в местные кооперативы

Direct:
  - Google Maps: "cooperativa agricola [город]" → собери контакты
```

**Вопросы:**
```
1. Как часто подаёте CAP-заявки? (frequency)
2. Сколько платите gestoras за подачу? (цена)
3. Были ли отказы или задержки? (pain)
4. Какие документы нужны? (scope)
5. Готовы платить €400/год за автоматизацию? (WTP)
6. Используете WhatsApp для бизнеса? (channel)
```

**Цель:** 15-20 интервью, 70%+ скажут "yes, готов платить"
