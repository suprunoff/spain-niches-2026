## Scoring Formula

```
Score = (D×20 + G×20 + R×15 + S×15 + M×10 + A×10 + F×5 + T×5) / 100

D = Demand (search vol, pain)
G = Gap (competitor void)
R = Revenue potential (LTV/CAC)
S = Spain tailwind (regulation, culture)
M = MVP speed (weeks to launch)
A = Founder access (customer ease)
F = Defensibility (moats)
T = Timing (urgency)
```
