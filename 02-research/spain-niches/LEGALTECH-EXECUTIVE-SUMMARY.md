## ТОП-3 Нихи (Рекомендация)

### 1️⃣ AI Finiquito Calculator (Score: 8.1/10)

**Суть:** Бесплатный калькулятор выходного пособия + консультация с адвокатом.

| Параметр | Значение |
|----------|----------|
| **TAM** | €50-100M/год |
| **Pain Level** | 🔴 КРИТИЧЕСКАЯ (люди теряют 5-20% от положенного) |
| **MVP Time** | 3-4 недели |
| **Barrier to Entry** | 🟢 LOW |
| **Конкуренция** | 🟢 LOW (10+ бесплатных калькуляторов, но все простые) |
| **Monetization** | Freemium (calc) → €15-30 консультация → €99 subscription |
| **GTM** | SEO + Google Ads + lawyer referrals |

**Why Start Here:**
- Самый быстрый MVP (3-4 нед)
- Нет технических барьеров (не нужна госинтеграция)
- Ясный монетизационный путь (lead gen для абогадов)
- Низкий CAC (органический поиск + реферралы)

**Validation Checklist:**
- [ ] 20 интервью (уволенные): 70%+ скажут "急迫" (urgent)
- [ ] 50%+ готовы платить €15-30 за консультацию
- [ ] 5+ абогадов согласны быть партнёрами для lead-gen
- [ ] Landing page конвертит 2%+ из трафика

### 2️⃣ Digital Due Diligence Inmobiliaria (Score: 8.3/10)

**Суть:** One-stop платформа для проверки юридической чистоты недвижимости.

| Параметр | Значение |
|----------|----------|
| **TAM** | €120-240M/год (самый большой!) |
| **Pain Level** | 🔴 КРИТИЧЕСКАЯ (3+ часа работы на сделку) |
| **MVP Time** | 6-8 недель |
| **Barrier to Entry** | 🟡 MEDIUM (нужен скрап + анализ) |
| **Конкуренция** | 🟢 LOW (нет интегрированного решения) |
| **Monetization** | B2B (Idealista, Fotocasa) + B2C (€200-500/сделка) |
| **GTM** | B2B partnerships + direct to lawyers + agents |

**Why Parallel with #1:**
- Выше TAM (€120-240M vs €50-100M)
- Может быть B2B (партнёры: Idealista, Fotocasa, банки)
- Sticky product (5+ сделок/год per lawyer)
- Less competition than Finiquito (no established players)

**Validation Checklist:**
- [ ] 15 интервью (abogados недвижимость): 80%+ скажут "экономил бы 3+ часа"
- [ ] 10 интервью (real estate agents): 60%+ готовы платить €200-500/сделка
- [ ] Registry API доступен (legal cover for scraping)
- [ ] 5-10 takers за €200-500
