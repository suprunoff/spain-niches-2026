## Spain-специфика (важно!)

### Платежи
- **Bizum:** Обязателен для consumer (80%+ используют)
- **Tarjeta/Redsys:** Для онлайн платежей компании
- **Transferencia SEPA:** B2B, крупные суммы

### Коммуникация
- **WhatsApp = деловой мессенджер** (40-60% reply rate vs 0.5-1.5% email)
- **Личные связи > всего** — warm intro = 8-10x выше cold
- **Facebook-группы:** Хосты, подрядчики собираются в 50+ активных групп (100K+ членов)

### Язык
- **Испанский ОБЯЗАТЕЛЕН** (100% пользователей)
- **Каталанский:** +20% CR в Барселоне

### Потребительское поведение
- **85% пробуют бесплатно** — free trial essential
- **92% проверяют отзывы** перед покупкой
- **72% используют промокоды** — цена важна

## Red Flags

🚩 **Не трогать:**
- **Нижесредний TAM** (<€20M) + сложный MVP (>14 недель) = не масштабируется
- **Никто не платит** (freemium only, нет paid conversion)
- **Высокий CAC** (>€300 за клиента) без LTV >= €2000
- **Строгая регуляция** без простого compliance пути
- **Network effect обязателен**, но нет способа холодного старта

## Быстрый старт

### Неделя 1
1. Выбрать одну из Top 3 ниш
2. Запустить 5-10 customer interviews (WhatsApp + Facebook-группы)
3. Проверить: есть ли готовность платить?

### Неделя 2-3
1. Начать MVP планирование (tech stack, API интеграции)
2. Запустить Google Ads/Facebook ads для валидации спроса (€100-200 budget)
3. Составить список konkurentов, их слабости

### Неделя 4-8
1. Лаунч MVP (минималист feature set)
2. Рекрут ранние юзеры через Facebook-группы (free trial)
3. Собрать feedback, итерировать

### Месяц 3+
1. Монетизация (trial → paid)
2. Масштабирование через Google Ads + partnerships с платформами
3. Добавление фич based на user feedback
