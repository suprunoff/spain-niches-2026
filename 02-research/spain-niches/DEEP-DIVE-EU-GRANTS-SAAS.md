## EXECUTIVE SUMMARY

**Ниша:** AI-платформа для автоматизации поиска, скрининга и подачи заявок на европейские гранты и субсидии для испанских SMB и консультантов.

**Тезис:** €27B неиспользованного финансирования NextGenerationEU + дедлайн 31 августа 2026 + 58% грантов не присуждаются + консультанты берут 15–25% = ненасыщенный рынок с готовой платёжной аудиторией.

**Вывод:** **GO (7.2/10)** — подтвержденный спрос, низкая конкуренция в сегменте автоматизации, высокий LTV/CAC через партнёрства с консультантами, максимальная срочность до августа 2026.

## 1. РАЗМЕР РЫНКА: ДЕНЬГИ ЕСТЬ

### 1.1 NextGenerationEU: €27B+ неиспользованного финансирования

**Ситуация на март 2026:**

| Параметр | Цифра | Примечание |
|----------|-------|-----------|
| **Выделено Испании** | €77.2B | Общее выделение NextGen |
| **Получено к Q1 2026** | ~€73B | Т.е. освоено 95% объёма |
| **Переведено в реальный сектор** | €45.95B | 63% от полученного |
| **Ещё ЧЕ НЕ достигло бизнеса** | **€27–31B** | КРИТИЧЕСКОЕ ОКНО |
| **Дедлайн** | 31 августа 2026 | 5 месяцев до конца |
| **Потенциал консультантских комиссий** | €4.05B (15% × €27B) | Рынок, на который можешь претендовать |

**Источник:** [El dinero público que millones podrían pedir pero España devuelve a Europa](https://www.elconfidencialdigital.com/articulo/dinero/ayuda-publica-que-millones-podrian-pedir-espana-devuelve-europa-usar/20250625200754970194.html), [El 58% de las subvenciones convocadas están sin conceder](https://www.elindependiente.com/economia/2025/07/07/58-subvenciones-convocadas-fondos-europeos-sin-conceder/)

### 1.2 Почему деньги не раздаются?

**Главные причины невыполнения (58% грантов не присуждены):**

1. **Бюрократия** (77% жалоб)
   - Формы на 50+ страниц
   - Требования неясны
   - Разные критерии для разных программ

2. **Незнание** (30%+ SMB)
   - Владельцы малого бизнеса не знают о программах
   - Нет простого инструмента "Какие гранты мне подходят?"
   - Минимальная осведомлённость о Kit Digital, PERTE, Horizonte Europa

3. **Сжатые дедлайны** (32% жалоб)
   - Каждый грант имеет разные даты подачи
   - Нет централизованного календаря
   - Часто на подготовку даётся 2–4 недели

4. **Риск отклонения** (неявно)
   - Если не соответствуешь критериям → потеря времени
   - Нет быстрого способа проверить соответствие до подачи

**Результат:** 300K+ компаний участвуют в NextGen проектах, но большинство получает только 1–2 гранта, хотя им доступны 5–15 программ.

**Источник:** [Las pymes sólo han recibido el 10% de los fondos europeos](https://emprendedores.es/ayudas/pymes-fondos-europeos/), [Burocracia y desconocimiento: barreras](https://www.directivosyempresas.com/pymes/burocracia-y-desconocimiento-las-barreras-en-la-gestion-de-subvenciones-de-los-fondos-europeos-para-las-pymes/)

### 1.3 Kit Digital: Доказательство спроса

**Самая успешная программа NextGen в Испании:**

- **860,000+ помощей выданных** (до конца 2025)
- Бюджет: €3K–€29K на компанию (разные сегменты)
- **ВСЕ ЕЩЁ ОТКРЫТА ДО 2026** (пока есть деньги)

**Критический инсайт:** 860K компаний успешно получили финансирование, потому что программа ПРОСТАЯ:
- 1. Выбираешь тип решения
- 2. Выбираешь провайдера из списка одобренных
- 3. Получаешь деньги

**Обратная сторона:** Остальные 3M+ SMB в Испании не знают о других €27B.

**Источник:** [Kit Digital España estadísticas](http://espanadigital.gob.es/lineas-de-actuacion/programa-kit-digital)

### 1.4 Другие программы с неиспользованным финансированием

**Horizonte Europa (R&D):** €15B+ для Испании, но только 5–10% SMB участвуют

**PERTE (Strategic Projects):** €4.5B+ через:
- Automotive (electric vehicles): €2.5B
- Aerospace: €2.1B
- Agribusiness & Hydrogen: €1–2B каждая

**Целевая аудитория:** Manufacturers, suppliers, R&D centers.

**Региональные программы (18 сообществ):** €5–10B ещё не освоено
- Каталуния (Cataluña): €2.5B
- Мадрид: €1.8B
- Страна Басков: €1.2B
- Остальное: €1–2B

### 2.2 Приватные платформы (основные конкуренты)

#### A. FANDIT (близко к монополии в discovery)

**Профиль:**
- **Основана:** 2015
- **Уникальное предложение:** 47,000+ субсидий в БД, alerts, фильтры по сектору/CNAE/бюджету
- **Модель:** Freemium + Premium (цена неизвестна публично)
- **Масштаб (оценка):** 10K–50K активных пользователей (оценка на основе BDNS посещаемости)

**Сильные стороны:**
- Лучшая база данных субсидий в Испании
- Удобный интерфейс для поиска
- Email alerts
- Фильтры по профилю

**Слабые стороны:**
- **НЕ автоматизирует подачу заявок** (только поиск)
- **НЕ проверяет соответствие критериям** (компания сама должна понять)
- **НЕ интегрирован с консультантами** (нет маркетплейса)
- Не хватает AI-скрининга
- Premium цена неизвестна, но судя по индустрии — €20–50/мес

**Источник:** [FANDIT - Buscador de subvenciones](https://fandit.es/), [Blog FANDIT](https://blog.fandit.es/)

#### B. Infoayudas.com

**Профиль:**
- 1,300+ текущих программ
- Бесплатный поиск
- Минимальная функциональность

**Угроза:** НИЗКАЯ (мало юзеров, мало фич)

#### C. Konsultoria.online + Projekta Innovación

**Профиль:**
- Enterprise-ориентированные
- Success fee модель
- Цена: €10K–50K за проект

**Угроза:** СРЕДНЯЯ (не конкурируют в SMB сегменте)

### 2.3 Консультанты по субсидиям (партнёры или конкуренты?)

**Основной источник спроса, но ручной работы:**

| Компания | Модель | Кого обслуживают | Комиссия |
|----------|--------|------------------|----------|
| **Eurosubvenciones** | Success fee only | SMB + Enterprise | 15–20% |
| **ECSA Consultores** | Success fee | 2,000+ клиентов | 10–15% |
| **MOVA** | Success fee + advisory | AI-интеграция (реклама) | 12–18% |
| **Rubvex** | Tiered (15% до €100K, затем ↓) | Generic SMB | 3–15% |

**Цифры:**
- €100K–500K грант: консультант берёт **15–20%** = €15K–100K за одного клиента
- €500K–1M: **10–15%** = €50K–150K
- €1M+: **3–10%** = €30K–300K

**Ключевой инсайт:** Консультант, который экономит 40% времени на форм-филлинг, готов платить €200–500/мес за SaaS. Это твой CAC.

**Источник:** [Tarifa consultoría subvenciones](https://www.solucionesong.org/consulta/que-porcentaje-sobre-la-subvencion-concedida-suele-cobrar-un-redactor-de-proyectos/3810/view)

## 3. БОЛЕВЫЕ ПУНКТЫ: ПОТРЕБИТЕЛЬ ГОТОВ ПЛАТИТЬ

### 3.1 Боль для SMB (потребители)

**Типичная ситуация:**
> "Я владелец SaaS B2B, работаю 5 лет, у меня 8 сотрудников. Вчера читал статью про грант на AI для стартапов. Заколебался: как узнать, подхожу ли я? Какие документы нужны? Сколько времени займёт заявка? Консультант требует €3K авансом, я не рискую. В результате я вообще не подаю заявку и теряю €20K–100K."

**Боль:** Неопределённость + стоимость информации + страх не подходить.

**Решение:** За 5 минут узнать все доступные гранты, за 2 часа заполнить большую часть документов.

### 3.2 Боль для консультантов (партнёры)

**Текущий workflow:**
1. Клиент приходит → "найди мне гранты"
2. Консультант вручную сканирует BDNS, FANDIT, региональные порталы (3–5 часов)
3. Консультант заполняет формы от руки (5–10 часов)
4. Клиент рецензирует, правки (2–3 часа)
5. **Итого: 10–18 часов на одного клиента × €60–100/час = €600–1800 внутренних затрат**

**Боль:** Низкая маржа из-за ручной работы. 50% времени уходит на "дура" работу.

**Решение:** AI-скринер находит гранты за 2 минуты, автозаполнение сокращает ручную работу с 10 часов до 3–4 часов → маржа ↑ 40%.

### 3.3 Почему именно СЕЙЧАС (срочность)

**Дедлайн 31 августа 2026 = МАКСИМУМ срочности:**
- Остаётся 5 месяцев
- Консультанты в панике (клиенты звонят в спешке)
- Правительство сокращает дедлайны (давление сверху)
- **Психология:** Люди платят ЗА СРОЧНОСТЬ

**Психологический эффект:** "Упущу 27B → больше не будет" → готовность платить СЕЙЧАС.

## 4. БИЗНЕС-МОДЕЛЬ: ВЫБИРАЕМ ВАРИАНТ

### Вариант A: Freemium B2C (SMB как конечные пользователи)

**Механика:**
- Бесплатно: поиск грантов, 1 сканирование в месяц
- Pro (€29–49/мес): неограниченный скан, автозаполнение, compliance check
- Enterprise (€299–999/мес): белый ярлык для консультантов, API доступ

**Плюсы:**
- ✅ Простая монетизация
- ✅ Масштабируется на всех 3M+ SMB
- ✅ Низкий чёрнили (€29)

**Минусы:**
- ❌ Высокий CAC (нужна реклама Google Ads)
- ❌ Низкий LTV (freemium конвертирует 2–3%, чёрнили €29 = LTV €230 за год)
- ❌ Низкий conversion rate (спрос неявный)
- ❌ Август 2026 = конец спроса (дедлайн)

**Unit Economics:**
- CAC: €80–150 (Google Ads)
- LTV: €230 (год) или €690 (3 года)
- LTV/CAC ratio: 1.5–2.9 (ПЛОХО, нужна ≥3)

**Прогноз первый год:**
- 1000 signups (при €10K spend на ads)
- 2–3% to Pro = 20–30 paying users
- MRR: €600–1500 (СЛАБО)

**Вывод:** Вариант A = долгосрочный play, не подходит для срочности августа 2026.

---

### Вариант B: B2B для консультантов (Partner модель) ⭐ РЕКОМЕНДУЕТСЯ

**Механика:**
- Консультанты платят €299–999/мес за SaaS
- Они получают: BDNS scraper, AI questionnaire, autofill, белый ярлык
- ТЫ получаешь: ежемесячную подписку + revenue share (3–5% от их success fee)

**Дополнительный доход (revenue share):**
- Партнёр выигрывает грант €500K
- Партнёр берёт комиссию 10% = €50K
- Ты получаешь 3% от их €50K = €1,500
- Если партнёр делает 5 таких грантов в год = €7,500 дополнительного дохода

**Плюсы:**
- ✅ CAC НИЗКИЙ (€150–300, через личные контакты, LinkedIn)
- ✅ LTV ВЫСОКИЙ (€299–999/мес × 24 мес = €7K–24K)
- ✅ LTV/CAC: 23–160x (ОТЛИЧНО)
- ✅ Уже готовая платёжная аудитория (консультанты привыкли платить)
- ✅ Эффект масштаба: 500 консультантов × €500/мес = €250K MRR

**Минусы:**
- ❌ Меньше пользователей (500 консультантов vs 3M SMB)
- ❌ Нужна BD (продажи) на консультантов
- ❌ Дедлайн август 2026 может сократить спрос (но консультанты всё равно будут работать)

**Unit Economics:**
- CAC: €150–300
- LTV: €7,200–24,000 (24-месячный контракт)
- LTV/CAC ratio: 24–160x (ОТЛИЧНО)
- Revenue share: дополнительно €3K–10K/партнёра в год

**Прогноз первый год:**
- 50 партнёров к концу года (5–10 в месяц)
- 50 × €500/мес = €25K MRR
- Revenue share: ~€5K/мес дополнительно
- **Итого: €30K MRR = €360K/год**

**Вывод:** Вариант B = ЛУЧШИЙ выбор. Быстро к прибыльности, высокий LTV/CAC, партнёры = маркетинг.

---

### Вариант C: Success Fee (платишь, если грант одобрен)

**Механика:**
- Клиент не платит ничего, пока не получит грант
- Ты берёшь 2–3% от суммы гранта
- Пример: помог выиграть €500K → ты получаешь €10K–15K

**Плюсы:**
- ✅ Нулевой CAC (нет возражений на цену)
- ✅ Очень высокий LTV (один грант €500K = €10K доход)
- ✅ Выравнивание интересов (тебе нужен результат)

**Минусы:**
- ❌ Кассовый разрыв (ты работаешь 3–6 месяцев, деньги приходят потом)
- ❌ Риск: грант может быть отклонён (ты потеряешь время)
- ❌ Конкуренция с консультантами (они тоже берут success fee)
- ❌ Сложно масштабировать (нужна большая команда на отклонения)

**Unit Economics:**
- CAC: €0 (но скрытая стоимость в отклонениях)
- LTV: €10K–30K на клиента (за 1–2 гранта в год)
- Кассовый разрыв: 3–6 месяцев

**Вывод:** Вариант C = долгосрочный play, но требует капитала для кассового разрыва. Не подходит для MVP.

## 5. ТЕХНИЧЕСКИЙ MVP: ЧТО РЕАЛЬНО?

### 5.1 Стек и сроки

**Фазы разработки:**

#### Фаза 1: Discovery (Недели 1–2)
- **BDNS API/Scraper**
  - Parse https://www.pap.hacienda.gob.es/bdnstrans
  - Извлечение: название, бюджет, дедлайн, критерии, регион
  - Данные: ~5K текущих + история
  - Сложность: НИЗКАЯ (открытые данные, HTML parsing)
  - Время: 2–3 дня

- **FANDIT API/Scraper**
  - Интеграция с FANDIT API (если открыт) или веб-скрапинг
  - Дублирование данных: нет, объединение
  - Сложность: СРЕДНЯЯ (FANDIT может закрыть доступ)
  - Время: 3–4 дня

- **Региональные БД** (опционально для MVP)
  - Каталуния: https://seu.gencat.cat/ca/web/hisenda/subvencions
  - Мадрид: https://www.comunidad.madrid/...
  - Сложность: СРЕДНЯЯ (разные форматы)
  - Время: 1 неделя (skip для MVP)

**Tech stack:**
- Backend: Python/FastAPI (быстро) или Node.js/Express
- Frontend: React/Next.js (для dashboard)
- DB: PostgreSQL (временные ряды scores)
- Scraping: BeautifulSoup/Selenium (Python)

**Сроки:** 5–7 дней

---

#### Фаза 2: AI Questionnaire (Недели 1–2)
- **Вопросник (5–10 вопросов)**
  - "Сколько сотрудников?" (чтобы определить SMB/PYME)
  - "Сектор деятельности?" (CNAE)
  - "Основной вид затрат?" (R&D, инфраструктура, обучение)
  - "Регион?" (для региональных программ)
  - "Бюджет, который тебе нужен?" (€5K–100K? €100K+?)
  - "Приоритет?" (срочно? долгосрочно?)

- **AI Matching (Claude/Gemini)**
  - Системный prompt: "Ты эксперт по субсидиям в Испании. Дано описание компании. Найди ТОП-10 релевантных грантов из БД."
  - Input: JSON с ответами на вопросник
  - Output: [{ grantId, matchScore: 0.85, reason: "..." }, ...]
  - Сложность: НИЗКАЯ (API call)
  - Время: 2–3 дня

**Tech stack:**
- Claude API или Gemini API
- Prompt engineering (100 строк)

**Сроки:** 3–4 дня

---

#### Фаза 3: Email/WhatsApp Alerts (Неделя 2)
- **Email notifications**
  - Крон-джоб, проверяет BDNS каждый день

- Revenue share: €12,000

- **Итого: €7,000/мес**

**Маржа: €37,950 - €7,000 = €30,950 MRR = профитабельно**

**COGS на единицу:**
- API calls (BDNS, email, WhatsApp): €2–5 на пользователя в месяц
- Infra: €10 на пользователя в месяц
- **Итого COGS: €12–15/пользователя (низко)**

**Gross margin: 85%+ (типично для SaaS)**
