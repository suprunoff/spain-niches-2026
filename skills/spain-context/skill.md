# Spain Market Context

## Обязательный чеклист для любого анализа ниши

Перед финальным выводом проверь каждый пункт через призму Испании:

### 1. Язык и локализация
- [ ] Продукт ОБЯЗАН быть на **испанском (castellano)**
- [ ] Для Каталонии → опционально каталонский
- [ ] Поддержка на испанском — минимальный стандарт
- [ ] Английский только как fallback для expat-сегмента

### 2. Платежи
- [ ] **Bizum** — обязателен для consumer (30.6M пользователей)
- [ ] **Redsys/TPV** — для карточных платежей (не Stripe напрямую)
- [ ] **Transferencia bancaria** — для B2B сегмента
- [ ] Подписки работают, но нужно явное доказательство ценности

### 3. Регуляторика (актуально на 2026)
| Закон | Дедлайн | Кого касается |
|-------|---------|---------------|
| VeriFactu (e-invoicing) | **Янв 2027 (SL)**, Июл 2027 (autónomos) | Все бизнесы кроме SII-обязанных |
| Ley Atención Cliente (Ley 10/2025) | Дек 2026 | Только 250+ сотр / €50M+ выручка |
| Tourist rental (VUD) | Июл 2025 | Арендодатели |
| BIM | С 2024 | Стройка (публичные) |
| RGPD/LSSI | Всегда | Все |

### 4. Бизнес-культура
- **Личные связи > cold outreach** — нужны introductions или LinkedIn warm approach
- **WhatsApp = деловой канал** — даже B2B. Поддержка через WhatsApp Business обязательна для SMB-сегмента
- **Принятие решений медленнее** — у SMB цикл 2–4 недели, у корпораций 3–6 месяцев
- **Лояльность высокая** — если зашёл, остаётся надолго

### 5. Ценообразование
- **€1,400/мес** — средняя зарплата. Pricing должен быть адаптирован
- Типичный SaaS для autónomo: €20–60/мес
- Типичный SaaS для PYME: €100–500/мес
- Корпоративный: €500–2,000/мес
- **60%+ испанцев чувствительны к цене** — freemium или trial критичен

### 6. Регионы
| Регион | Особенности |
|--------|-------------|
| Madrid | Бизнес-хаб, корпорации, tech startups |
| Barcelona / Cataluña | Tech экосистема, каталонский язык, регуляторные нюансы |
| Valencia | SMB, туризм, агро |
| Andalusia | SMB, туризм, медленнее внедряют tech |
| Islas Canarias | Туризм, VUD критичен, tax benefits (ZEC) |
| País Vasco | Промышленность, высокий доход, Euskera |

### 7. Цифровые привычки
- **80%+ трафика с мобильных** — mobile-first обязателен
- **Google Reviews критичны** — 4.0+ = доверие
- **Тактика доверия:** testimonials, логотипы клиентов, числа

### 8. Go/No-Go метрики (проект-специфичные)
| Метрика | Минимум | Цель |
|---------|---------|------|
| Google поиск/мес (ES) | 500 | 1,000–5,000 |
| Конкурентов | 5 | 10–30 |
| CPC | €1.00 | €2.00+ |
| "Urgent problem" в интервью | 70%+ | 90%+ |
| Готовность платить | 50%+ | 70%+ |
| Niche score | 6.0 | 7.5+ |
