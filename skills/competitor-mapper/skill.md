# Competitor Mapper — Spain Market

## Цель
Построить конкурентную карту ниши: кто играет, как позиционируются, где дыры.

## Процесс (30–60 мин)

### Шаг 1: Discovery (10 мин)
Найди всех игроков через:
- Google ES: `[ниша] software/app/herramienta`
- Product Hunt: поиск по категории
- Capterra ES / GetApp ES
- `ads-competitor` скилл → Meta/Google Ad Libraries
- App Store / Google Play (если mobile)

Цель: список из 10–30 игроков.

### Шаг 2: Классификация

```markdown
## Категории игроков

### Прямые конкуренты (решают ту же проблему)
| Компания | URL | Год | Цена | Рейтинг | Кол-во отзывов |
|----------|-----|-----|------|---------|----------------|
| ...      | ... | ... | ...  | ...     | ...            |

### Косвенные конкуренты (решают через другой подход)
| Компания | Подход | Почему клиенты уходят |
|----------|--------|----------------------|

### Потенциальные заменители (клиент сейчас делает это руками)
| Способ | Почему неудобно |
|--------|----------------|
```

### Шаг 3: Deep Dive ТОП-3 конкурентов

Для каждого из ТОП-3:
```markdown
## [Название конкурента]

- **URL:**
- **Описание:** одна строка — что делает
- **Целевой сегмент:** кто их клиент
- **Ценообразование:** freemium / €X/мес / one-time
- **Сильные стороны:**
  -
- **Слабые стороны (из отзывов):**
  -
- **Жалобы в Google/Capterra Reviews:**
  -
- **Рекламная активность:**
  - Google Ads: да/нет, примерный бюджет
  - Meta Ads: да/нет
- **Испанская адаптация:**
  - Интерфейс на испанском: да/нет
  - Поддержка на испанском: да/нет
  - Bizum/Redsys: да/нет
```

### Шаг 4: Gap Map

```markdown
## Gap Analysis

| Функция/Атрибут | Конкурент 1 | Конкурент 2 | Конкурент 3 | GAP? |
|-----------------|-------------|-------------|-------------|------|
| Испанский UI    | ✓           | ✗           | частично    | ✓    |
| [фича 2]        | ...         | ...         | ...         | ...  |

## Наши возможности
- **Quick win:** [что можно сделать лучше за 1 неделю]
- **Positioning gap:** [незанятая позиция]
- **MVP differentiator:** [одна фича, которой нет у всех]
```

### Шаг 5: Competitive Score

```markdown
## Итог конкурентного анализа

- Кол-во прямых конкурентов: __
- Наличие доминирующего игрока: да/нет
- Средний рейтинг топ-3: __ / 5.0
- Топ-жалобы клиентов: [список]
- Наш angle входа: [позиционирование]
- Сложность входа: низкая / средняя / высокая
```

## Выход

Сохрани в `03-competitors/{niche-name}/competitive-analysis.md`

## Инструменты
- `ads-competitor` — Ad Library сканирование
- `perplexity-search` — быстрый обзор игроков
- `reddit-insights` — жалобы пользователей на конкурентов
- `deep-research` — детальный анализ конкретного игрока
