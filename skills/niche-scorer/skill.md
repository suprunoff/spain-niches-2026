# Niche Scorer — Spain Market

## Методология
7 критериев × веса = итоговый score (0–10).
Порог входа: **≥ 6.0** для deep dive, **≥ 7.5** для MVP.

## Критерии и веса

| # | Критерий | Вес | Описание |
|---|----------|-----|----------|
| 1 | Рыночный спрос | ×2 | Объём поиска 500–5K/мес ES, рост тренда |
| 2 | Монетизация | ×2 | CPC > €1, чёткая модель, LTV/CAC > 3x |
| 3 | Конкуренция | ×2 | 5–30 игроков, нет хорошо финансируемых |
| 4 | Регуляторный рычаг | ×1.5 | Compliance-driven = стикинесс и дедлайны |
| 5 | Испанская специфика | ×1.5 | Язык, Bizum, WhatsApp, региональность |
| 6 | MVP-реализуемость | ×1 | Сборка за 2–6 недель, solo или 2-member team |
| 7 | Личная синергия | ×1 | Экспертиза, сеть, интерес к теме |

**Формула:**
```
Score = (D×2 + M×2 + C×2 + R×1.5 + S×1.5 + V×1 + P×1) / 11
```

## Шаблон оценки

```markdown
## Ниша: [Название]

### Критерии (1–10)
- Рыночный спрос (D): __/10
  - Поиск/мес в ES: ~___
  - Тренд: ↑/→/↓
  - Источник: [ссылка]

- Монетизация (M): __/10
  - Модель: SaaS / marketplace / услуги
  - Ср. чек: €___/мес
  - CPC: €___
  - LTV/CAC оценка: ___x

- Конкуренция (C): __/10
  - Кол-во игроков: ___
  - Топ-конкуренты: [список]
  - Слабости топ-игроков: [что плохо]

- Регуляторный рычаг (R): __/10
  - Закон/дедлайн: [какой, когда]
  - Pain без решения: [описание]

- Испанская специфика (S): __/10
  - Продукт на испанском: да/нет
  - Bizum/Redsys нужен: да/нет
  - Региональная специфика: [какая]

- MVP-реализуемость (V): __/10
  - Срок MVP: ___ недель
  - Стек: [технологии]
  - Core feature: [одна строка]

- Личная синергия (P): __/10
  - Экспертиза: [есть/нет/прокачать]

### Итоговый Score
Score = (D×2 + M×2 + C×2 + R×1.5 + S×1.5 + V×1 + P×1) / 11 = **__/10**

### Вердикт
- [ ] < 4.0 → Пропустить
- [ ] 4.0–6.0 → Мониторинг
- [ ] 6.0–7.5 → Deep dive
- [ ] 7.5+ → MVP-кандидат

### Go/No-Go чеклист
- [ ] Поиск 500–5K/мес
- [ ] CPC > €1.00
- [ ] 5–30 конкурентов
- [ ] Есть платящий сегмент в Испании прямо сейчас
- [ ] Продукт реализуем за ≤6 недель
```

## Процесс скоринга

1. Запусти `perplexity-search` для быстрого обзора ниши (5 мин)
2. Заполни шаблон выше
3. Проверь Go/No-Go чеклист
4. Добавь нишу в `02-research/spain-niches/spain-niches-full-v2.md`
5. Обнови ТОП-список если score > 7.5

## Инструменты
- `perplexity-search` — быстрый обзор рынка
- `deep-research` — детальный анализ
- `ads-competitor` — проверка рекламного спроса
- `reddit-insights` — валидация боли у аудитории
- `niche-finder` — поиск gap в рынке
