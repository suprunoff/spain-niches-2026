## 💡 Key Strategic Insights

### Why Club Operations Wins
1. **Uncharged market:** No one solving club revenue optimization
2. **Clear pain:** Playtomic commission is quantifiable, avoidable
3. **Weak competition:** TPC expensive, Playtomic product-market misaligned
4. **High unit economics:** LTV/CAC 25x (vs industry 3-5x)
5. **Expanding TAM:** 900 clubs now, growing as sport grows

### Competitive Moat (Why Hard to Copy)
1. **Not replacing Playtomic:** Complementary (clubs stay with Playtomic)
2. **Domain-specific:** Dynamic pricing logic + padel metrics hard to copy
3. **Network effects (eventually):** Clubs using = data = better models
4. **Personal relationships:** SMB SaaS retention via great CS

### International Expansion Path
1. **Phase 1:** Spain (100 clubs, €12k MRR)
2. **Phase 2:** Portugal (30 clubs, €3.6k MRR)
3. **Phase 3:** France (40 clubs, €4.8k MRR)
4. **Total Year 3:** 500+ clubs, €27k+ MRR potential
